## Instalación

npm install

## Uso

Debe asegurarse de configuarar la base de datos mongodb en el archivo .env antes de 
realizar esto pasos.

Para iniciar el proyecto en modo producción:

npm run start

Para iniciar en modo desarrollador:

npm run dev
